0.)To make virtual env ---- python -m venv <name>

1.)  To start project -- django-admin startproject project_name

2.) To run server ---python manage.py runserver

3.) To create app --- python manage.py startapp

4.) To create susperuser --- python manage.py createsusperuser

5.)Add the made apps in settings

6.)--to add static files like css in codebase (add these in settings)---
STATIC_URL = 'static/'
STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]

7.) --to add templates, make folders name templates, you can make templates in each seperate
app folder for that respective app
